Diferencia entre JPG, PNG y GIFF
JPG:
Para que JPG tenga un tamaño reducido, el archivo comprime los datos
de la imagen en bloques de píxeles, también conocidos como “mosaico”
haciendo que pierda calidad.		 PNG:
Se creó para satisfacer las necesidades de aquellos que necesitaban
obtener imágenes de máxima calidad, cosa que el formato JPG no es capaz
de ofrecer.

Este formato de archivo permite la transparencia cosa que JPG no es capaz
de ofrecer. 		GIFF:
Este formato permite crear pequeños clips de vídeo a partir de varios archivos
de imagen, algo que no se puede hacer con otros formatos de imagen como
PNG o JPG.

Sólo admite 256 colores, por lo que no es un formato de archivo adecuado para
trabajar con fotografías de alta calidad y que tengan una amplia gama cromática.

¿Cuál es el mejor formato para la creación de páginas webs?
Según que tipo de imagen que quieras: Si quieres que sea una imagen con movimiento el mejor es GIFF ya que es el único formato de estos tres que permite el movimiento,
pero se tiene que tener en cuenta sus limitaciones de color (solo 256) y de transparencia (lo es toda la imagen o no lo es).
Si necesitas que se conserven los detalles de la imagen con detalle, precisión y la mejor precisión debes elegir el formato de archivo PNG. La siguiente decisión en este caso es seleccionar la paleta
de colores más adecuada, pues de ello dependerá que el tamaño del archivo sea más o menos grande. Si se necesitan menos de 256 colores tendrá que elegirse un PNG-8. Si es para una fotografía se usará el PNG-24.
En cualquier otro caso, la elección más adecuada es el formato JPG, aunque deberá experimentarse con las distintas opciones de compresión para llegar a un tamaño adecuado sin una pérdida apreciable de la calidad de la misma.
